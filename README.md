CODSOFT – Python Programming Internship
This repository contains the tasks completed as part of the CODSOFT Python Programming Internship. Each task is implemented using Python, focusing on core logic, clean structure, and user-friendly interfaces.

Tasks Included
Task 1 – Contact Book (GUI)
Add, view, update, and delete contacts
Persistent storage using JSON
Task 2 – Password Generator (GUI)
Generate secure passwords based on length and strength
Strength meter and copy-to-clipboard feature
Task 3 – Rock Paper Scissors Game (GUI)
Interactive game with target score and winning streak tracking
